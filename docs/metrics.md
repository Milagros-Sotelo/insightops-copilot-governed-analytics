# Governed metric catalog

The metric catalog is the semantic contract shared by Python, SQL, Power BI and Copilot. Metrics cannot be invented at question time. Versioned definitions should be reviewed by the named owner before their effective date.

The demo implements ten KPI: net sales, gross margin, expenses, orders, invoices, payments, active suppliers, average inventory, cycle time and compliance. Each result is linked to a period and its definition; budget variance is calculated only when a valid budget exists.
